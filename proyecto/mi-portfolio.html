Programando con Java

En el desarrollo formativo de Diseo de Aplicaciones Web (DAW) hemos realizado numerosos ejercicios con el lenguaje de Java.


Tras aprender la estructura y la lógica a implementar en la programación de forma genérica hemos llevado a cabo ejercicios relacionados con diferentes temas dentro del lenguaje de JAVA.
Desde métodos, arrays y Strings, con ejercicios tales como impkementar la simulación del juego del ahoracado, piedra, papel o tiejeras, etc. También hemos visto las clases y objetos, el manejo de excepciones, las estructuras de datos dinámicas y, por último, el acceso a ficheros.

Programando con PHP

En el módulo de Desarrollo en Entorno Servidor hemos impartido el lenguaje de PHP.


Hemos visto el procesamiento de formularios, las funciones y librerias y la gestión de sesiones y cookies.

Programando con JavaScript

En el módulo de Desarrollo en Entorno Cliente hemos dado el lenguaje de JavaScript.


Aunque a día de hoy seguimos impartiendo esta asignatura, ya hemos tocado temas como los operadores y la gramática respectiva al leguaje de programación impartido, los bucles, objetos y arrays y, por último, la funciones.
A día de hoy estamos viendo el funcionamiento y estructura genérica de JSON y cómo utilizar el select en formularios y mostrar un elemento en el HTML en función de la respuesta escogida por el cliente.

Desarrollo web con HTML y CSS

HTML junto con CSS es quizás lo que mayormente hayamos visto a lo largo de la formación, empezando en el primer curso con el módulo de Lenguaje de Marcas y terminando en segundo con Diseño de Interfaces (además del HTML y CSS requerido en los ejercicios de formularios de JS y PHP


Este es el primer proyecto como tal desarrollado en el curso. Se trata de una página web informativa donde el alumnado se presenta y muestra sus dotes tanto como diseñador de páginas web como el desarrollo de la misma.
